    🐳 Docker Compose

    Docker Compose est l'outil qui lance plusieurs conteneurs Docker en une seule commande. Le fichier docker-compose.yml définit désormais cinq services : fastapi, ollama, stablediffusion, postgres et mongo.

    Pour simplifier la vie du développeur, toutes les commandes utiles sont regroupées dans le Makefile.

    Commandes utiles : 

    # Démarrer les services en arrière-plan
make up

# Arrêter l'ensemble
make down
